# LaundryKu - Sistem Laundry Sederhana

Aplikasi web untuk mencatat order laundry kiloan.
Dibuat pakai HTML, CSS, dan JavaScript.
Data disimpan di LocalStorage browser, jadi gak perlu database.

## Fitur
- [x] Tambah, Edit, Hapus data order
- [x] Hitung otomatis total bayar berdasarkan berat & paket
- [x] Update status: Diterima → Proses
